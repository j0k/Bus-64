🔐 Online Crypter

Шифрование в браузере на Web Crypto API. Ничего не отправляется на сервер — весь код работает локально.

Ключ выводится из пароля через PBKDF2-SHA256 (210 000 итераций, случайная соль). Забыли пароль — расшифровать нельзя.
Формат: [версия][алгоритм][соль 16][iv][шифртекст](+HMAC) → base64. Работает офлайн.
